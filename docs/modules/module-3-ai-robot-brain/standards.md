# Documentation Standards for Module 3: The AI-Robot Brain (NVIDIA Isaac)

## Citation Format

All sources must follow APA (American Psychological Association) 7th edition formatting. Each claim, fact, or concept derived from external sources must include proper attribution.

### Examples:

- **Website**: Author, A. A. (Year, Month Date). Title of webpage. Site Name. URL
- **Book**: Author, A. A. (Year). *Title of work*. Publisher.
- **Journal Article**: Author, A. A. (Year). Title of article. *Title of Periodical*, volume(issue), pages. https://doi.org/xx.xxx/yyyy
- **Technical Documentation**: NVIDIA Corporation. (Year). *Isaac Sim documentation*. NVIDIA Developer. URL

## Source-Grounding Requirements

All technical content must be verified against authoritative sources:

1. **Official NVIDIA Isaac Documentation**: All Isaac Sim and Isaac ROS concepts must align with official NVIDIA documentation
2. **Academic References**: Where possible, include academic papers on robotics, AI, and humanoid systems
3. **Industry Best Practices**: Reference established best practices for robotics simulation and AI implementation
4. **Minimal 40% Source Coverage**: At least 40% of content should be supported by official documentation or academic references

## Content Structure Standards

### Document Format
- Use Docusaurus-compliant Markdown
- Include frontmatter with title and sidebar position
- Use consistent heading hierarchy (H1 for document title, H2 for sections, etc.)
- Include relevant code blocks with proper language tags

### Technical Accuracy
- Verify all technical claims against official documentation
- Include example code that is tested and verified
- Provide troubleshooting sections for common issues
- Include diagrams and illustrations where beneficial

## Review Process

All content must undergo:
1. Technical accuracy verification by domain experts
2. Source verification to ensure proper attribution
3. Reproducibility checks for all examples and exercises
4. Compliance verification against module requirements
